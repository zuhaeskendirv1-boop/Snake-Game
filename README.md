# Snake-Game

A classic Snake game built in C++.
This was my first project to learn Object-Oriented Programming (OOP).

## How to run it
1. Download the files
2. Open a terminal in the project folder
3. Run: g++ GameForm.cpp -o snake
4. Run: ./snake

## Controls
- Arrow keys to move the snake
